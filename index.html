HTML

Hypertext Markup Language

Har bir website HTML strukturasi asosida quriladi, hoh u yuqori vazifalarni bajaruvchi online magazin website bo'lin yo oddiy shaxshiy web sahifa bo'lsin

CSS

Cascading Style Sheets

Agar HTML website ga struktura bergan bo'lsa CSS unga bezak va yaxshi ko'rinish berib o'tadi. Misol tariqasida agar inson suyagi HTML bo'lsa terisi CSS.

JS

JavaScript
